Modelos de Representación del Color

¿Por qué el RGB falla en Robótica?

En ingeniería mecatrónica a menudo necesitamos que un sistema identifique una pieza por su color (ej. "clasificar tapas rojas"). Si usamos el espacio de color RGB (Red, Green, Blue), una simple sombra o cambio de iluminación alterará los tres canales simultáneamente, rompiendo nuestro algoritmo.

La solución es convertir la imagen al espacio cilíndrico HSV (Hue, Saturation, Value), el cual aísla el color real de la intensidad lumínica.

El Cono del Espacio HSV

V (Brillo)

El color puro gira alrededor del círculo (H). La cantidad de color (S) va del centro al borde. La iluminación (V) es un eje vertical que va del negro (abajo) al color máximo (arriba).

H: Matiz (Hue)

Define el color puro o pigmento (Rojo, Verde, Azul, etc.). Se representa como un ángulo en el cilindro.

¡Trampa de OpenCV!

En teoría H va de 0° a 360°. Para que quepa en un byte (8 bits max 255), OpenCV divide el ángulo a la mitad. En OpenCV, H va de 0 a 179.

S: Saturación (Saturation)

Define la pureza o intensidad del color. Va desde el gris/blanco descolorido (0) hasta el color vívido puro (255).

0 (Gris)255 (Puro)

V: Valor (Value / Brillo)

Representa la intensidad de la luz que ilumina la pieza. 0 es negro absoluto (oscuridad) y 255 es el brillo máximo (luz plena).

Extracción de un color mediante InRange

# -*- coding: utf-8 -*-
import cv2
import numpy as np

img = cv2.imread('piezas.jpg')

# 1. Convertir imagen BGR (Default OpenCV) a HSV
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

# 2. Definir los límites del color Azul (H~120)
# Orden: [Hue (0-179), Saturation (0-255), Value (0-255)]
limite_inferior = np.array([100, 150, 50])
limite_superior = np.array([140, 255, 255])

# 3. Crear máscara binaria (Píxeles dentro del rango = 255, resto = 0)
mascara = cv2.inRange(hsv, limite_inferior, limite_superior)

# 4. Operación BITWISE AND para mostrar el color original extraído
resultado = cv2.bitwise_and(img, img, mask=mascara)

cv2.imshow('Original', img)
cv2.imshow('Mascara Binaria', mascara)
cv2.imshow('Piezas Azules', resultado)
cv2.waitKey(0)

Color Tracking Dinámico por Webcam

Tu objetivo es detectar y enmascarar un objeto sólido de color vívido (por ejemplo, una tapa de botella azul o roja, un post-it, etc.) utilizando la cámara de tu laptop.

Instrucciones Paso a Paso:

  1. Calibración: Toma una foto con tu cámara al objeto que vas a rastrear. Cárgala en la pestaña Simulador y juega con los sliders hasta aislarlo perfectamente en la máscara blanca. Anota los límites.
  2. Código: Copia el script de la derecha en tu IDE local (VS Code, PyCharm o Spyder).
  3. Configuración: Reemplaza lower_color y upper_color en el código con los valores de tu calibración.
  4. Ejecución: Corre el programa, muestra el objeto a la cámara e intenta moverlo rápidamente para comprobar la robustez del filtrado HSV frente a sombras.
import cv2
import numpy as np

# 1. Iniciar captura de video. 0 indica la cámara por defecto.
cap = cv2.VideoCapture(0)

while True:
    # Leer un frame (cuadro) de la cámara
    ret, frame = cap.read()
    if not ret: break

    # 2. Espejar frame (opcional para UX natural) y convertir a HSV
    frame = cv2.flip(frame, 1)
    hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)

    # 3. REEMPLAZA ESTOS VALORES CON LOS DE TU SIMULADOR
    lower_color = np.array([100, 150, 50])
    upper_color = np.array([140, 255, 255])

    # 4. Construir máscara y aplicar Bitwise
    mask = cv2.inRange(hsv, lower_color, upper_color)
    res = cv2.bitwise_and(frame, frame, mask=mask)

    # 5. Mostrar las tres ventanas
    cv2.imshow('Original', frame)
    cv2.imshow('Mascara Binaria', mask)
    cv2.imshow('Objeto Detectado', res)

    # Salir presionando la tecla 'q' o 'ESC' (27)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

# Liberar hardware y cerrar ventanas
cap.release()
cv2.destroyAllWindows()
