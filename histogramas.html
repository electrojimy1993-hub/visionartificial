Análisis de Histogramas y Contraste

El histograma es la "firma estadística" de la iluminación de una imagen. En visión artificial industrial, se utiliza para diagnosticar la exposición de la cámara, segmentar objetos mediante umbrales y mejorar drásticamente el contraste a través de técnicas de ecualización global o ecualización adaptativa (CLAHE).

Python 3.x cv2.calcHist cv2.equalizeHist

1. El Histograma

Es un gráfico de frecuencias que muestra cuántos píxeles existen para cada nivel de intensidad de luz (0 a 255). Matemáticamente:

$$h(r_k) = n_k$$

# Cálculo en OpenCV
hist = cv2.calcHist([img],[0],None,[256],[0,256])

2. Ecualización Global

Mapea los valores de intensidad utilizando la Función de Distribución Acumulada (CDF) de toda la imagen para intentar "aplanar" el histograma y estirar el contraste.

$$s_k = T(r_k) = (L-1) \sum_{j=0}^{k} p_r(r_j)$$

# Mejora global de contraste
img_eq = cv2.equalizeHist(img)

3. CLAHE (Adaptativa)

Contrast Limited Adaptive Histogram Equalization. Divide la imagen en bloques (tiles) para evitar sobreexposición local y recorta el ruido con un límite clipLimit.

# Configuración adaptativa
clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8,8))
img_cl = clahe.apply(img)

Código de Inicialización y Gráficas

En Python, se utiliza matplotlib para renderizar la gráfica del histograma calculada por OpenCV. Copia este script base: 

# -*- coding: utf-8 -*-
import cv2
import numpy as np
from matplotlib import pyplot as plt

# 1. Cargar imagen en escala de grises
img = cv2.imread('radiografia_industrial.jpg', cv2.IMREAD_GRAYSCALE)

# 2. Aplicar CLAHE (Recomendado para imágenes con mucho ruido)
clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8,8))
resultado = clahe.apply(img)

# 3. Calcular histograma del resultado
hist = cv2.calcHist([resultado], [0], None, [256], [0, 256])

# 4. Mostrar resultados visuales y gráficos
cv2.imshow('Original vs Procesada', np.hstack((img, resultado)))

plt.plot(hist, color='gray')
plt.title('Histograma CLAHE')
plt.xlim([0, 256])
plt.show()

cv2.waitKey(0)
cv2.destroyAllWindows()
