Introducción Práctica

Operaciones de Bajo Nivel en Imágenes

En mecatrónica, el procesamiento de imágenes es crucial para que los robots y sistemas automatizados entiendan su entorno. Antes de aplicar Inteligencia Artificial compleja, las imágenes se filtran, segmentan y transforman mediante operaciones aritméticas (control de iluminación, fusión de sensores), operaciones lógicas/bit a bit (enmascaramiento de regiones de interés) y transformaciones geométricas (corrección de perspectiva de cámaras y alineación).

Python 3.x OpenCV (cv2) NumPy

1. Operaciones Aritméticas

A diferencia del desbordamiento en NumPy (250 + 10 = 4), OpenCV emplea aritmética saturada (250 + 10 = 255). Esto evita artefactos visuales indeseados.

# Suma con saturación
res = cv2.add(img1, img2)

# Fusión con pesos (Blend)
res = cv2.addWeighted(img1, alpha, img2, beta, gamma)

Útil para: Ajustar brillo, fusión de imágenes y desvanecidos (cross-fade).

2. Operaciones Bit a Bit

Trabajan a nivel de bits de cada píxel de forma independiente. Son de gran utilidad cuando queremos aislar regiones irregulares mediante una máscara binaria (píxeles blancos de interés, negros de fondo).

# Enmascaramiento común
res = cv2.bitwise_and(img1, img1, mask=mascara)

# Inversión de imagen
res = cv2.bitwise_not(img1)

Útil para: Aislar objetos de interés, aplicar logos, y segmentación cromática.

3. Operaciones Geométricas

Modifican la disposición espacial de los píxeles. Las transformaciones afines de OpenCV utilizan matrices de transformación de tamaño $2 \times 3$ para trasladar, rotar y escalar.

# Rotación con matriz 2D
M = cv2.getRotationMatrix2D(centro, angulo, escala)
res = cv2.warpAffine(img, M, (ancho, alto))

Útil para: Enderezado de cámaras, estabilización y aumento de datos (Augmentation).

Código de Inicialización en Python

Para comenzar la práctica real en tu computadora, instala las librerías necesarias con pip install opencv-python numpy y arranca con este código base para leer y preparar las imágenes: 

# -*- coding: utf-8 -*-
import cv2
import numpy as np

# 1. Cargar las imágenes de prueba en escala de grises
# Nota: Para operaciones aritméticas y de bits, deben tener las mismas dimensiones
imagen_a = cv2.imread('imagen_a.png', cv2.IMREAD_GRAYSCALE)
imagen_b = cv2.imread('imagen_b.png', cv2.IMREAD_GRAYSCALE)

# Redimensionar si es necesario para asegurar tamaños idénticos
ancho, alto = 400, 400
imagen_a = cv2.resize(imagen_a, (ancho, alto))
imagen_b = cv2.resize(imagen_b, (ancho, alto))

# Aquí aplicamos las operaciones que experimentarás en la pestaña del "Simulador"
# Por ejemplo, suma ponderada:
resultado_blend = cv2.addWeighted(imagen_a, 0.7, imagen_b, 0.3, 0)

# Mostrar resultados
cv2.imshow('Resultado', resultado_blend)
cv2.waitKey(0)
cv2.destroyAllWindows()
