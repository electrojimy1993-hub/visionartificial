Color Tracking y Extracción de Coordenadas

Filtrar una imagen por color (como vimos en HSV) nos da una máscara binaria. Pero para que un robot actúe, necesita coordenadas espaciales ($X, Y$). El Tracking (seguimiento) extrae matemáticamente los contornos de esa máscara para calcular el centro de masa del objeto y trazar cajas delimitadoras (Bounding Boxes) en tiempo real.

1. Detección de Contornos

Un contorno es una curva que une todos los puntos continuos a lo largo de un límite que tienen el mismo color o intensidad. OpenCV utiliza el algoritmo de Suzuki para extraer la topología de la máscara binaria.

# Extraer contornos de la máscara
contornos, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

2. Centroides (Momentos)

Para calcular el centro de gravedad de una forma, usamos los Momentos de la Imagen. El centroide $(C_x, C_y)$ se obtiene dividiendo los momentos espaciales de primer orden por el momento de área.

$$C_x = \frac{M_{10}}{M_{00}} \quad ; \quad C_y = \frac{M_{01}}{M_{00}}$$

3. Bounding Box & Filtrado

Aislamos el ruido evaluando el área de cada contorno (ignorar manchas pequeñas). Si supera el umbral, dibujamos un rectángulo que encierre el objeto.

# Rectángulo Delimitador
x, y, w, h = cv2.boundingRect(contorno)
cv2.rectangle(img, (x,y), (x+w, y+h), (0,255,0), 2)

Estructura Básica del Algoritmo de Tracking

# 1. Asumiendo que ya tienes una 'mascara' generada por cv2.inRange()...
contornos, jerarquia = cv2.findContours(mascara, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

for c in contornos:
    area = cv2.contourArea(c)
    
    # 2. Filtro de Ruido: Solo procesar objetos grandes (Área > 500 px)
    if area > 500:
        # 3. Obtener momentos para calcular el centro exacto
        M = cv2.moments(c)
        if M["m00"] != 0:
            cx = int(M["m10"] / M["m00"])
            cy = int(M["m01"] / M["m00"])
            
            # Dibujar el centro (Punto rojo)
            cv2.circle(frame, (cx, cy), 5, (0, 0, 255), -1)

        # 4. Trazar Bounding Box (Rectángulo verde)
        x, y, w, h = cv2.boundingRect(c)
        cv2.rectangle(frame, (x, y), (x+w, y+h), (0, 255, 0), 2)

cv2.imshow('Color Tracking', frame)

Seguimiento por Servomotor (Pan/Tilt)

Conecta una webcam sobre un servomotor controlado por un Arduino. Utiliza Python (PySerial) para calcular el "error de posición" del objeto y enviar un comando de control para que la cámara gire e intente mantener el objeto siempre en el centro de la pantalla.

Teoría de Control Básico (Control Proporcional):

  1. SetPoint (Referencia): El centro de tu pantalla (e.g., $X = 320$ si tu cámara es 640x480).
  2. Variable Medida: La coordenada $C_x$ (Centroide) de tu objeto extraída mediante momentos espaciales.
  3. Cálculo de Error: $\text{Error} = C_x - \text{SetPoint}$.
  4. Señal de Control: Envía el error al Arduino para mover el Servo: si el error es positivo, mueve el servo a la derecha; si es negativo, a la izquierda.
